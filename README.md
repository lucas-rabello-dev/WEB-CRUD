### Resumo:
Um projeto pessoal feito para por em prática minhas habilidades com Golang servindo arquivos estáticos e templates html localmente e também integrando formulários html com banco de dados.

O foco é ser um projeto back-end deixando um pouco de lado a parte visual do site.


### Como executar:
Na raiz do projeto rode execute com:
```go 
go run cmd/main.go
```
